template<typename REAL_T , std::size_t dim_x, std::size_t dim_u, std::size_t dim_p, typename... Args>
using hybrid::OdeFunction = typedef std::function< void(std::array< REAL_T, dim_x > &dx, REAL_T t, const std::array< REAL_T, dim_x > &x, const std::array< REAL_T, dim_u > &u, const std::array< REAL_T, dim_p > &p, Args...) >

Type for an ODE function callback.

The type is a prototype for the RHS of an ODE callback, it tries to follow the same scheme of the functions as required by the parametric estimation toolbox in MATLAB.

\[ \dot{x}(t) = f(t, x(t), u(t), p), \quad t \in \mathbb{R}_+, \, x(t) \in \mathbb{R}^{\dim(x)}, \, u(t) \in \mathbb{R}^{\dim(u)}, \, p \in \mathbb{R}^{\dim(p)} \]

The actual implementation follows this calling convention:

f(dx, t, x, u, p, ...)
Parameters
dxthe output of the function
tthe current time
xthe current state
uthe current input
pthe paramenter vector
Template Parameters
REAL_Tthe floating point precision type
dim_xsize of the unknown vector
dim_usize of the external input vector
dim_psize of the parameter vector
Argsvariadic user arguments

Detailed Description

Author
Matteo Ragni

The library implements a Runge-Kutta 4 scheme with the following tableau:

0 | 0 0 0 0
1/2 | 1/2 0 0 0
1/2 | 0 1/2 0 0
1 | 0 0 1 0
----+----------------
| 1/6 1/3 1/3 1/6

For the ode in the form:

\[ \dot{x}(t) = f(t, x(t), u(t), p), \quad x(t_0) = x_0 \]

The integration scheme evaluates:

\[ x(t_k + h) = x(t_k) + t_s \sum_{i = 1}^{4} b_i k_i \]

where:

\[ k_i = f(t + c_i t_s, x(t_k) + t_s \sum_{j=1}^{i-1} a_{i,j} k_j, u(t_k), p) \]

The input \( u(t_k) \) is assumed constant between integration ministeps. From an implementation point of view, \( p \) is an array of parameters This is done for compatibility with MATLAB System Identification Toolbox model file. No allocation is performed.

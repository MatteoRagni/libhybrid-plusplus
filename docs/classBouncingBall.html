Detailed Description

The actual bouncing ball class.

This is the class that the user has to implement in order to use the library. The class implements (as required per implementation):

  • a flow map
  • a jump map
  • a jump set
  • a flow set
  • an output map
  • a flow jacobian Futhermore the class need to override (as for now, will be removed in future releases) the prepare_callbacks method in order to instruct the integrator.

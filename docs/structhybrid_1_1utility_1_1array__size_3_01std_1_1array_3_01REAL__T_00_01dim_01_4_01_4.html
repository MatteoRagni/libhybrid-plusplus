Detailed Description

template<typename REAL_T, std::size_t dim>
struct hybrid::utility::array_size< std::array< REAL_T, dim > >

Get dimension of an array compile time.

The constexpr can be used as follows:

template<typename T, std::size_t S>
inline void f(std::array<T, S>) {
std::cout << hybrid::utility::array_size<T, S>::size << std::endl;
}
std::array<float, 10> a;
f<float, 10>(a);
// prints:
// 10

It is intended use is in static assertion in order to get compilation consistent code (for example in dgels, we need to check at compile time the size of the matrix and the array)
